# Sales Dashboard Analysis

## Project Overview
This project analyzes global sales data to identify trends, top-performing products, regional performance, and business insights using Data Analytics techniques.

## Objectives
- Analyze sales performance
- Identify profitable regions and categories
- Create interactive dashboard
- Generate business insights

## Tools Used
- Python
- Pandas
- SQL
- Power BI
- Excel

## Project Structure
- data/raw → Original dataset
- data/processed → Cleaned dataset
- notebooks → Jupyter notebooks
- sql → SQL queries
- dashboard → Power BI dashboard
- screenshots → Dashboard images
- reports → Business insights
- presentation → Project presentation

## Dataset
Global sales dataset containing:
- Orders
- Sales
- Profit
- Region
- Category
- Customer data

## Expected Outcomes
- Interactive sales dashboard
- KPI analysis
- Trend analysis
- Business recommendations
